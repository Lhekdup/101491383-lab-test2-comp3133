## Overview
This Angular application fetches and displays Harry Potter character data using the public HP-API. It features a character list, a filtering system by Hogwarts House, and a detailed view for individual characters using Angular Material components.

## How to Run This Project Locally
1. Clone the repository.
2. Run `npm install` to install the required dependencies.
3. Run `ng serve` to start the development server.
4. Open your browser and navigate to `http://localhost:4200/`.
